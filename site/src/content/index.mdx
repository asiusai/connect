# Better Openpilot Experience

Better file management, nicer UI, video streaming, joystick control, and more for your openpilot device.

We have three components: Connect, Forks and API - use them together or pick what you need.
You can try our [upgraded Connect](https://connect.asius.ai) right now - works with stock openpilot, just log in with your Comma, Konik, or Asius account.

## AsiusPilot

Modified openpilot with remote streaming, joystick control, and device params editing.

Reset your device and then input [installer.asius.ai](https://installer.asius.ai)

## Asius API (in alpha)

Cloud backend with 1TB storage, fast video playback, and no device blocking.

After installing our fork, enable "Asius API" from the device settings, then use [Asius Connect](https://connect.asius.ai) going forward.

[More info](/api)

## FAQ

<Details title="How much does it cost?" client:load>
  Everything is free for now. Asius API will have an monthly fee once it's stable and feature-complete.
</Details>

<Details title="Which devices are supported?" client:load>
  Any device that can run openpilot (comma 3, comma 3X, comma four, konik A1M, etc.).
</Details>

<Details title="Is the code open source?" client:load>
  Yes, all code is hosted on [GitHub](https://github.com/asiusai/asiusai) under MIT licence. Pull requests are welcome!
</Details>

<Details title="I found a bug, what do I do?" client:load>
  [Report it on GitHub Issues](https://github.com/asiusai/asiusai/issues). This project is actively developed by one person while supporting 3 APIs, 2
  openpilot forks, our own connect API, and multiple devices - so bugs happen. Please include your device, fork, and API setup when reporting.
</Details>

<Details title="I have an idea or question" client:load>
  [Join the Discussions](https://github.com/asiusai/asiusai/discussions) on GitHub.
</Details>

<Details title="What's being worked on?" client:load>
  Check the [Project Board](https://github.com/orgs/asiusai/projects/1/views/1) to see planned features and current progress.
</Details>
